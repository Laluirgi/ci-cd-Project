# Sistem Otomatisasi Pengingat Tugas Berbasis Rule-Based

Proyek ini adalah sistem pengingat tugas otomatis menggunakan aturan (rules) sederhana.

## Fitur
- Rule-based engine untuk menentukan pengingat
- Notifikasi otomatis
- CI/CD Pipeline lengkap menggunakan GitHub Actions

## Teknologi
- Node.js
- Express.js
- Docker
- GitHub Actions (CI/CD)

## Cara Menjalankan Secara Lokal

```bash
# Clone repository
git clone https://github.com/Laluirgi/-Sistem-Pengingat-Tugas-CI-CD-Project.git

# Masuk ke folder
cd -Sistem-Pengingat-Tugas-CI-CD-Project

# Install dependencies
npm install

# Jalankan aplikasi
npm start
